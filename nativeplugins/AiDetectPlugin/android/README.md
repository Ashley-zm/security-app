# Android AAR output

Build the Android library module that contains `com.example.aidetect.AiDetectPlugin`,
then place the generated release AAR in this directory.

Recommended output name:

```text
AiDetectPlugin-release.aar
```

The final local native plugin package used by HBuilderX should contain:

```text
nativeplugins/AiDetectPlugin/package.json
nativeplugins/AiDetectPlugin/android/AiDetectPlugin-release.aar
```
